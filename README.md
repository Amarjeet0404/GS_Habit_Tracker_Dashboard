# 📊 Habit Tracker Dashboard

A comprehensive Google Sheets-based Habit Tracker Dashboard designed to help users build consistency, track daily habits, monitor progress, and analyze performance through automated calculations and visual insights.

---

## 🚀 Overview

The Habit Tracker Dashboard enables users to:

* Track daily habits using interactive checkboxes.
* Set completion goals for each habit.
* Monitor progress through completion percentages.
* Visualize performance with progress bars.
* Rank habits based on consistency and achievement.
* Analyze weekly and monthly habit trends.

This project demonstrates the use of spreadsheet automation, dashboard design, and data visualization techniques using Google Sheets.

---

## ✨ Features

### ✅ Daily Habit Tracking

* Checkbox-based habit completion system.
* Track habits across a monthly schedule.
* Supports multiple habits simultaneously.

### 📅 Weekly Organization

* Monthly tracker divided into weekly sections.
* Easy identification of habit consistency patterns.

### 📈 Progress Analytics

Automatically calculates:

* Completed Goals
* Remaining Goals
* Completion Percentage
* Progress Indicators

### 🏆 Habit Ranking

* Sort habits based on completion percentage.
* Identify top-performing habits.
* Highlight habits that need improvement.

### 🔄 Automated Calculations

Utilizes Google Sheets formulas such as:

* `COUNTIF()`
* `SUM()`
* `SORT()`
* `FILTER()`
* `VLOOKUP()`
* Percentage calculations

### 🎨 Dashboard Visualization

* Progress bars
* Percentage indicators
* Structured and user-friendly layout
